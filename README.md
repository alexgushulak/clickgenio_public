## To run the client:

Run this command inside the client directory:
- npm install
- npm run dev

To open the local instance of the website, enter:
- -o

Links to libraries:
- https://mui.com/material-ui/getting-started/
- https://react.dev/


## To run the server:

Run this command inside the server directory:
- npm install
- npm run server
